Project Description : This project is a Profile Management System developed using Java technologies including HTML, JSP, Servlet, JavaBean, JDBC, and Database integration following the MVC (Model-View-Controller) architecture.
The system allows users to manage profile information by storing and retrieving data from a database.
System features : Add new student profile. Save student information. View all saved student information. Delete student information. Edit student information.
